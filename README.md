# Inizio case study

A tool for getting the first page of Google search results as data on request

## Scope of work

- Create a single input that accepts the query text
- Receive the first page from Google search as non-html data
- Write unit tests for the code
- Upload and submit work


## Quick start

#### `npm install`:

Install the dependencies.

#### `npm run dev`:

Run the app, you can view the app by visiting `http://localhost:1234/`.
